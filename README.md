# RepositorioCpp
Estructuras de datos en c++

Se tiene las estructuras:

- Lista con Arreglos 
- Lista con Nodos (Lista enlazada)
- Pilas con Arreglos
- Pilas con Nodos
- Colas con Arreglos
- Colas con Nodos
- Colas circulares (pendiente)
- Arbol con Arreglos
- Arbol con Nodos
